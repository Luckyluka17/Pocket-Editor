Pocket Editor

Pocket Editor est un outil de développement léger. Il permet aux développeurs de
pouvoir éditer leur code Python & de l'executer. Avec sa boîte à outil Python,
vous trouverez les ressources necessaires à l'amélioration de votre code et,
elle est aussi pratique pour les débutants.
